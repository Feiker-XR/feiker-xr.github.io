Redis的简介及安装和命令行操作.

缓存系统

缓存(Cache)是基于Model层和数据库之间的。可以缓解甚至接管数据库的压力.

Redis的定义

Redis(Remote Dictionary Server),是一个开源的有 ANSI C语言编写的,遵循BSD协议,支持网络,可基于内存亦可以持久化的日志型、key-value数据库。

Redis的优势

  • 操作具有原子性(保证事务的完整性)
  • 常驻内存,读写性能优越(远胜于硬盘IO),读11万次/s,写8.1万次/s
  • 支持多种数据格式
  • 可以自动保存数据到硬盘,服务重启时恢复

Redis数据类型

  • String: 字符串
  • Hash: 哈希散列
  • List: 列表
  • Set: 集合
  • Sorted Set: 有序集合
